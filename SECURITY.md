# Security Policy

| Version | Supported |
|---------|-----------|
| Latest  | ✅ |

## Reporting a Vulnerability

**Please do not open a public GitHub issue for security vulnerabilities.**

Contact the maintainer directly:

👉 **[github.com/joemugen](https://github.com/joemugen)** — send a message via GitHub profile

Include in your report:
- A description of the vulnerability
- Steps to reproduce
- Potential impact
- Any suggested fix (optional)

You will receive a response within **72 hours**. If the issue is confirmed, a patch will be released as quickly as possible and you will be credited in the release notes.
