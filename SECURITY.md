## Reporting a Vulnerability

**Please do not open a public GitHub issue for security vulnerabilities.**

Report security issues privately via GitHub's security advisory system:
**[Report a vulnerability](https://github.com/joemugen/obsidian-cli-mcp/security)**

> If the link above shows a 404, "Private vulnerability reporting" may not be enabled yet on this repository. In that case, contact the maintainer directly via [GitHub](https://github.com/joemugen).

Include:
- A description of the vulnerability
- Steps to reproduce
- Potential impact
- Any suggested fix (optional)

You will receive a response within **72 hours**. If the issue is confirmed, a patch will be released as quickly as possible.
